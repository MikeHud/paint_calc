WordPress Embed Code

Paste the following into a Custom HTML block in WordPress. The iframe will automatically resize to fit the calculator as content expands.

<iframe
  id="paint-calc"
  src="https://mikehud.github.io/paint_calc/"
  width="100%"
  scrolling="no"
  frameborder="0"
  style="border:none; display:block; width:100%;">
</iframe>
<script>
window.addEventListener('message', function(e) {
  if (e.data && e.data.iframeHeight) {
    document.getElementById('paint-calc').style.height = (e.data.iframeHeight + 32) + 'px';
  }
});
</script>
