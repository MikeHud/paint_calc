Integration Tips

1. Path Configuration

Make sure to update the src attribute to point to the correct path where your calculator files are located.

2. Responsive Design

Use the responsive embedding method for better mobile experience. The calculator automatically adapts its styling when embedded.

3. Cross-Origin Issues

If embedding from a different domain, ensure CORS policies allow iframe embedding.

4. Performance

The calculator is lightweight and loads quickly. Consider lazy loading for better page performance.

5. Accessibility

The calculator includes proper ARIA labels and keyboard navigation for accessibility compliance.

API Integration

You can also integrate the calculator programmatically:

// Access the calculator API const calculator = window.JotunPaintCalculator; // Get paint data const paintData = calculator.paintData; // Trigger calculation programmatically calculator.calculateMixing(); // Listen for calculation events document.addEventListener('calculationComplete', function(event) { console.log('Calculation completed:', event.detail); });
